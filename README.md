
# Data Analytics & Business Intelligence — Task 1
## Maincrafts Technology | Internship June 2026

### Overview
End-to-end business analytics project analyzing 9,994 retail 
transactions from the Sample Superstore dataset to generate 
actionable business insights.

### Tools Used
- Microsoft Excel — Data Cleaning, EDA, Pivot Tables
- SQLite (DB Browser) — SQL Data Extraction & Validation
- Power BI Desktop — Interactive Dashboard
- 
## Dataset Used-
**Superstore Sample:**
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
### Deliverables
- Cleaned Excel file with 5 Pivot Tables
- SQL query file with 4 business queries
- Power BI Dashboard (.pbix)
- Business Insights Report (PDF)

### Key Findings
- Total Revenue: $2.30M | Total Profit: $286.40K
- West region leads at 31.58% revenue share
- Technology drives 50.8% of total profit
- Discounts above 20% consistently produce negative profit (SQL-validated)
- 18.72% of transactions generated a loss
- November is peak month | February is lowest

### SQL Queries Written
1. Total Sales by Region
2. Profit by Category
3. Monthly Sales Trend
4. Discount Impact on Profit Analysis

### Dashboard Features
- KPI Cards (Total Sales, Total Profit)
- Monthly Sales Trend Line Chart
- Top 5 Products Bar Chart
- Regional Sales Pie Chart
- Category Distribution Pie Chart
- Top 10 Customers Table
- Interactive Slicers (Date, Region, Category)
